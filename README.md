# Projeto-One---Conversor-de-moedas

# 💱 Conversor de Moedas - Projeto ONE

Este projeto foi desenvolvido como parte do programa **Oracle Next Education (ONE)**, com objetivo de praticar lógica de programação, Java e consumo de APIs.

---

## 🚀 Objetivo
Converter valores entre diferentes moedas de forma simples e rápida, usando taxas de câmbio atualizadas.

---

## 🛠️ Tecnologias Utilizadas
- **Java**
- **IntelliJ IDEA**
- **APIs para cotação de moedas**
- **Git & GitHub**

---

## ✨ Funcionalidades
✔️ Conversão entre múltiplas moedas  
✔️ Interface simples e intuitiva  
✔️ Cotações em tempo real  
✔️ Cálculos automáticos sem complicação
